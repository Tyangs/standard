# @tyangs/config

English / [简体中文](./README_CN.md)

## Usage

### Install

```bash
npm install @tyangs/config -D
# or
yarn add @tyangs/config -D
# or
pnpm install @tyangs/config -D
```

### tsconfig.json

```json
{
  "extends": "@tyangs/config/tsconfig.base.json"
  // Add your custom config here
}
```
